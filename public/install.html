Installation

Install Aithernet.

Get a node running on Ubuntu in a few steps. Packages are delivered through the authenticated portal once your access is approved.

Requirements

  • Ubuntu 24.04 LTS (amd64)
  • An approved early-access account
  • For radio work, a supported software-defined radio (optional to start)

Download from the portal

Sign in to the authenticated portal and download your signed package, checksums, and verification key from the Downloads page.

Open the portal

Verify the package

Verify the signature and checksums with the published key before installing. The portal shows the exact commands for your release.

Install

sudo apt update
sudo apt install ./aithernet_1.0.0~beta.3_amd64.deb

Run setup

Guided setup configures the node and, in hosted mode, enrolls it to your account.

aithernet setup

Start using Aithernet

Run your first mission in plain language.

aithernet run "<what you want done>"
Need help? See the documentation for setup options, provider configuration, enrollment, and troubleshooting.

The current early-access release is 1.0.0-beta.3. The portal always provides the exact authorized filenames and verification key for your release.
