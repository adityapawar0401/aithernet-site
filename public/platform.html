Layers

What each layer does.

Mission orchestration

Accepts objectives, plans the work, and tracks every mission through its lifecycle with budgets, approvals, and observable state.

Agent runtime

Runs coordinator and coding agents in isolated workspaces, using the providers you configure.

Tool and coding execution

Executes code and tool calls — including GNU Radio flowgraphs — through a managed runtime with bounded resources.

Node and mesh infrastructure

Manages node identity, enrollment, fleet health, and the meshes that authorize communication.

SDR transport

Carries authenticated peer messages over software-defined radio, with reservation, acknowledgement, retry, and delivery semantics.

Data and observability

Records events, artifacts, and mission outcomes, and produces reproducible datasets with consent.
