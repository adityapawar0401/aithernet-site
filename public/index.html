An autonomous SDR node runtime, operated as a product.

Aithernet runs a software-defined-radio node with a GNU Radio MCP backend, a coordinator/coding-agent layer with deterministic budgets and approvals, and a hosted control plane for enrollment, fleet health and signed releases — with privacy-first, consent-gated telemetry.

What it is

SDR node runtime

A local node that drives an SDR through a pinned, signed GNU Radio MCP component over stdio. Receive-first; transmission is disabled by default.

Hosted control plane

Outbound-only nodes enroll to a hosted control plane for identity, fleet health, heartbeats and signed release distribution. The node's local API is never exposed to the Internet.

Agent layer

Optional coordinator (Gemini API / OpenAI-compatible / local endpoint) and coding agent, each in an isolated workspace with deterministic budgets and explicit approvals. Never exposed to anonymous users.

Privacy-first telemetry

Telemetry is consent-gated and data-minimized. See the privacy & legal page.
